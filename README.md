# Название проекта
**Cats**
Карточки с упаковками корма для кота, которые можно выбирать для покупки


## Технологии
В проекте использованы следующие свойства html,css и js
 * Grid Layout
 * адаптивная верстка
 * классы JS
 * модули JS
 * webpack
